🔒 Security & Audit

Smart Contract

ZBANK is a standard Jetton on TON. The contract has been deployed using the open-source minter.ton.org tool. Ownership has been revoked — no one can mint new tokens, freeze wallets, or change metadata.

Audit Status

While ZBANK itself is a standard Jetton without custom code, the underlying TON blockchain and minter tool are audited and widely used. For additional security, we recommend storing ZBANK in non-custodial wallets (Tonkeeper, MyTonWallet).
